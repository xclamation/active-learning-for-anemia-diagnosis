# Active Learning for Anemia Diagnosis

Репозиторий содержит код и обученные модели для дипломной работы **«Методы активного обучения для диагностирования анемии»**.

Основная цель - показать, что методы активного обучения позволяют улучшить качество диагностирования анемии при значительной экономии ресурсов на разметку данных.
## Структура репозитория

| Файл                                              | Описание                                                                                                                       |
| ------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| `experiments.ipynb`                               | Код экспериментов: сравнение стратегий активного обучения (Least Confidence, Margin, Entropy, QBC) и обучение baseline‑модели. |
| `active_learning_pipeline_least_confidence.ipynb` | Готовый пайплайн метода на основе Least Confidence.                                                                            |
| `random_forest_baseline.joblib`                   | Базовая модель (Random Forest), обученная на всей доступной выборке.                                                           |
| `random_forest_final_al_least_conf.joblib`        | Финальная модель после 25 итераций активного обучения (Least Confidence).                                                      |
| `data_scaler.joblib`                              | Стандартизатор (StandardScaler), обученный на тренировочных данных.                                                            |
## Основные результаты

- **Бета‑талассемия**: Precision 0.91, Recall 0.91, F1‑score 0.91 (против 0.88 у baseline).  
- **Железодефицитная анемия**: F1‑score 0.89 (на уровне baseline).  
- **Общая макро‑F1**: 0.66 (baseline - 0.64).  
- **Экономия разметки**: использовано 700 примеров вместо 23 457 (в 33 раза меньше).
